# Python Website Blocker with Pattern Detection

A Python-based Website Blocker application that allows users to block and unblock websites by modifying the system hosts file. The project also includes basic web scraping and pattern detection functionality to automatically identify and block websites containing harmful or distracting content.

This project demonstrates Python file handling, web scraping, pattern recognition, and system-level automation concepts.

Important Notes:

- Run VS Code or terminal as Administrator
- Website blocking requires permission to edit the hosts file
- This project currently supports Windows systems only.

---

# Features

- Block websites manually
- Unblock previously blocked websites
- Automatic website scanning using web scraping
- Pattern detection based on keywords
- Detect harmful or distracting website content
- Command-line interactive interface

---

# Technologies Used

- Python
- Requests
- BeautifulSoup
- File Handling
- Web Scraping
- Pattern Recognition
